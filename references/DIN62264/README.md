# DIN 62264 Reference Files

This folder contains DIN 62264 standard documentation and reference materials.

## About DIN 62264

DIN 62264 is the German/international equivalent of ISA-95, defining a framework for integrating enterprise and manufacturing execution systems. It provides the foundation for developing this TypeDB schema ontology.
